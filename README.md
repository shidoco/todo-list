# Todo List
A simple project showcasing a list of tasks.

## Installation
First, make sure you have Node.js and npm installed on your computer.

1. Clone the repository:

    ```bash
    git clone <repository-url>
    cd todo-list
    ```
2. Install the dependencies:

   ```bash
   npm install
   ```

### Development Server
To run the development server, run the following in your terminal:

   ```bash
   npm run dev
   ```
Then open the app in your browser at the URL shown in the terminal: http://localhost:5173.
